# Builder Certification Framework

### Level 1: Operator
**"I can run, observe, and explain what a swarm is doing."**
* **Required Labs:** 0, 1, 2.
* **Evidence:** Screenshot of a Hive-Mind task phases; explanation of task assignment; memory persistence proof.

### Level 2: Architect
**"I can design, modify, and harden swarms for production use."**
* **Required Labs:** 3, 4.
* **Evidence:** Before/After YAML modification; Failure injection log; Learning analysis.

### Level 3: Systems Designer
**"I can design systems that remain defensible over time."**
* **Required Labs:** 5 + Capstone.
* **Evidence:** Completed ADR; Alternatives analysis; Dissent record.
